# F303 blank ioc
This repository contains an .ioc file to be used for **EVERY** STMF303K8 project of PERLA

# How to use
1. Create an empty stm32cubeIDE project
2. Remove all created files
3. Copy this .ioc file into the folder
4. Rename the file using STM32CubeIDE (Keep update reference on)
5. Generate code
6. Well done!
