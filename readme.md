# F303 blank ioc
This repository contains an .ioc file to be used for **EVERY** STMF303K8 project of PERLA

# How to use
1. Create a blank STMF303K8T6 project inside STM32CubeIDE
2. Replace generated *project_name.ioc* with this file
3. Congrats you just ensured that PERLA CAN will not be fucked up!
